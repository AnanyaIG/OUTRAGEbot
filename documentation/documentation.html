OUTRAGEbot\Script::on($event, $handler)

Description

Called to bind an event handler to this script.

Accepted arguments

  1. string $event Event name
  2. callback $handler Callback

Examples

$this->on("message", function($channel, $user, $message)
{
    if($channel == "#android")
        $user->message("you sent a message to #android");
});
$this->on("PRIVMSG", function($event)
{
    var_dump($event);
});

OUTRAGEbot\Script::addCommandHandler($command, $handler)

Description

Called to bind a command handler to this script.

Accepted arguments

  1. string $command Command to listen to
  2. callback $handler Callback for this command handler.

$this->addCommandHandler("run", "startRunning");

OUTRAGEbot\Script::getInstance()

Description

Retrieves the current connection controller - or, Connection\Instance.
You will need this if you're wanting to do such things as grab the username
of the bots (as an example...)
The method call can either be used or you can use the 'instance' property
from within a script.

Accepted arguments

This method does not take any arguments.

Examples

$this->addCommandHandler("whoami", function($channel, $user, $payload)
{
    $channel->message("It's me, your friend, ".$this->instance->socket->nickname."!");
});

OUTRAGEbot\Script::addTimer($callback, $interval, [ $repeat = 1, [ $arguments ]])

Description

Create a timer, with the ability to fine tune how many times you want the timer
to be called.

Accepted arguments

  1. callback $callback Timer handler
  2. double $interval Interval in seconds (can support fractional seconds)
  3. integer $repeat How many times to repeat the call, or -1 for infinite
  4. array $arguments Array of arguments passed to the timer function.

Examples

$callback = function()
{
    echo "this will get called every half a second";
};

$this->addTimer($callback, 0.5, -1);
$callback = function($hero = "Bob")
{
    echo "this will get called every half a minute, with ".$hero." as my hero!";
};

$this->addTimer($callback, 30, -1, [ "Sarah" ]);

OUTRAGEbot\Script::setInterval($callback, $interval, [ $arguments ])

Description

Creates an interval, being called approximately every n seconds.

Accepted arguments

  1. callback $callback Timer handler
  2. double $interval Interval in seconds (can support fractional seconds)
  3. array $arguments Array of arguments passed to the timer function.

Examples

$callback = function()
{
    echo "this will get called every thirty seconds";
};

$this->setInterval($callback, 30);
